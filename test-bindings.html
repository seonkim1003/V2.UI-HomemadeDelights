🔧 Cloudflare Bindings Test Page

Test R2 and KV bindings for your gallery application

📋 How to Use

Click each test button to verify your Cloudflare Pages bindings are configured correctly. Make sure you've configured R2 and KV bindings in Pages → Settings → Functions → Bindings and redeployed your project.

1. Check Bindings Detection

Verify that R2 and KV bindings are detected by the Worker function.

2. Test R2 Operations

Test if R2 bucket is accessible and can perform list operations.

3. Test KV Operations

Test if KV namespace is accessible and can perform read/write operations.

4. Test Image Serving

List images in R2 and test if they can be served via the API.

5. Full Integration Test

Run all tests together to verify complete functionality.
